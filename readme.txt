An E-commerce sales mobile webapp to show live sales data on your iPhone/iPod/Android device. Works with WP e-commerce now, more will be added soon.

== Description ==
This plugins help you get real time sales data like number of sales made today, this week, this month, this year and all time etc on your android/iphone or any other mobile/table device as webapp. 

The main feature of this plugin is that you can navigate to any day in history to see sales made on that day, i.e. to see sales made on 22 Jan 2009, you can go to **All Time->2009->January->3rd week->22nd Jan**, This way you will be able to see number of orders and items sold on 22 Jan 2009.

Other feature included are:

*   Supports **multiple currency** symbol. (Dollar, Yen, Pound, Euro, Indian Rupee etc..)
*   Supports Two different **web-app templates**, One for great visual effect (default), and another for best performance on old mobile device or for slow data connection.
*   Supports **Custom URL**: you can configure webapp at your own url like (www.yoursite.com/mywebapp) 
*   Supports **QR Code**: To quickly open webapp on mobile, just scan the QR code on plugin settings page.
*   Shows **best selling products** on each page. If you are looking at year over view, you will see best selling items for that year and same for month view, week view etc..

== Installation ==
1. Install this just like any other wordpress plugin.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Go to settings->Show My Sales for configuration and information on how to use...

== Frequently Asked Questions ==
= Which time-zone settings being used by plugin? =
This plugin respects (follows) time-zone settings of wordpress it self, that you can change at settings->General.

= What E-commerce plugins are supported? =
As of now, only WP e-commerce plugin by getshopped.org, we are working on to add more plugin compatibility in future.. However, if you want to make priority request for e-comm plugin you are using, you can always do so by visiting our site and filling up contact us form.
